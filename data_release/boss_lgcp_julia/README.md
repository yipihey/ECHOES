# BOSS CMASS-South LGCP catalogs (Julia GraphGP engine)

Generated by `pipeline/build_boss_lgcp_catalogs.py` on the **Julia GraphGP engine, GPU** — the same
method as the JAX path (anisotropic `K(Δθ,Δz)` from a window-deconvolved Landy–Szalay measurement,
`kernel_from_K2d` → `generate_catalogs_from_kernel`), only the backend swapped.

The field is anisotropic in observed coordinates, so this uses the GraphGP.jl anisotropic covariance
port (`AnisoCov`), validated against the Python fork's `aniso.py` generate to **9.0e-15** (CPU) and
the Julia GPU path to **5.6e-8** vs CPU. Julia-GPU is the no-OOM path: at `n_cand ≈ 2.2M`, `k=30`,
pure-JAX-GPU OOMs on the 46 GB A6000 (it materialises the `(M,31,31)` tensor) while GraphGP.jl
generates at ~13 M pts/s.

Files: `cmass_south_lgcp_julia_<i>.npz` — `(ra, dec, z)` of LGCP-sampled galaxies per field draw.
Reproduce: `JAX_PLATFORMS=cpu python pipeline/build_boss_lgcp_catalogs.py --n-cand-factor 20 --n-samples 5 --device gpu`.
