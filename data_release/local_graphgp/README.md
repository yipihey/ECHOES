# Local-volume GraphGP fields (Julia backend)

Generated by `pipeline/build_local_graphgp_field.py` on the **Julia GraphGP engine**, on the real
2M++ catalogue. The local volume is real-space comoving (velocity-corrected), so the field is
**isotropic** — no anisotropic `K(Δθ,Δz)` is involved (that is a BOSS observed-coordinate concern).
The Julia engine here is a pure backend swap, validated against the JAX path:

- engine equivalence (JAX vs Julia, shared grid graph, same white noise): **L2-rel 3.1e-8**
- `field_posterior_vecchia` vs the dense JAX posterior (real 2M++ subsample, exact limit): **8.2e-6**

Files (each `delta` is `1+δ`, mean ~1, on an observer-centred cube; `box_mpc` = side):

- `local_2mpp_graphgp_prior_cube.npz` — 64³ lognormal Cox prior draw (fiducial galaxy kernel
  r0≈6 Mpc, σ²=0.8); strictly positive, mean 0.99, contrast 0.01–37.
- `local_2mpp_graphgp_inpaint_cube.npz` — 48³ conditional Matheron inpaint conditioned on the
  36,855 real 2M++ galaxies within d<150 Mpc.

Reproduce: `OMP_NUM_THREADS=8 JAX_PLATFORMS=cpu python pipeline/build_local_graphgp_field.py`.
The kernel here is a fiducial isotropic galaxy clustering scale; calibrating it from a measured
local ξ(r) and the per-galaxy n̄ is a separate scientific step (the engine validation is
kernel-agnostic).
